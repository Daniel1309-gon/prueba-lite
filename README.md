# prueba-lite

## Objetivo
Validar tus conocimientos y técnicas en el desarrollo de aplicaciones utilizando las tecnologías Python, Django, React y PostgreSQL. Aunque no está descrito en la prueba, ten en cuenta las buenas prácticas de 
código limpio. Aplica Atomic Design en el proyecto y se evaluará la calidad del código y la estructura de carpetas. Utiliza librerías de IA, Blockchain y otras librerías novedosas para proponer una funcionalidad adicional en este proyecto.
